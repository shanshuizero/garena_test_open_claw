# 前端深入：TypeScript 专项英语

## 1. interface
- 发音：/ˈɪntərfeɪs/
- 含义：接口类型
- 例句：Use an interface when you want to describe an object shape clearly.
- 翻译：当你想清晰描述对象结构时，可以使用 interface。
- 讲解：TS 基础术语。

## 2. type alias
- 发音：短语表达
- 含义：类型别名
- 例句：A type alias is useful for unions and mapped types.
- 翻译：类型别名很适合联合类型和映射类型。
- 讲解：和 interface 形成对照。

## 3. extends
- 发音：/ɪkˈstendz/
- 含义：继承；扩展
- 例句：This interface extends the base response type.
- 翻译：这个接口扩展了基础响应类型。
- 讲解：接口设计高频表达。

## 4. intersection type
- 发音：短语表达
- 含义：交叉类型
- 例句：An intersection type combines multiple type requirements.
- 翻译：交叉类型会合并多个类型要求。
- 讲解：TS 进阶核心概念。

## 5. mapped type
- 发音：短语表达
- 含义：映射类型
- 例句：Mapped types reduce duplication in complex models.
- 翻译：映射类型可以减少复杂模型中的重复定义。
- 讲解：很适合工程化表达。

## 6. conditional type
- 发音：短语表达
- 含义：条件类型
- 例句：Conditional types are powerful but can be hard to read.
- 翻译：条件类型很强大，但可能不容易阅读。
- 讲解：表达 trade-off 很自然。

## 7. infer
- 发音：/ɪnˈfɜːr/
- 含义：推断关键字
- 例句：The infer keyword helps extract nested types.
- 翻译：infer 关键字有助于提取嵌套类型。
- 讲解：TS 高级类型必学。

## 8. utility type
- 发音：短语表达
- 含义：工具类型
- 例句：Utility types like Partial and Pick can simplify model definitions.
- 翻译：像 Partial 和 Pick 这样的工具类型可以简化模型定义。
- 讲解：工作中很常见。

## 9. type guard
- 发音：短语表达
- 含义：类型守卫
- 例句：A type guard makes the code safer and easier to understand.
- 翻译：类型守卫让代码更安全，也更容易理解。
- 讲解：TS 运行时判断高频术语。

## 10. discriminated union
- 发音：短语表达
- 含义：可区分联合类型
- 例句：A discriminated union is great for modeling async request states.
- 翻译：可区分联合类型非常适合描述异步请求状态。
- 讲解：前端状态建模高频表达。

## 11. declaration file
- 发音：短语表达
- 含义：声明文件
- 例句：We may need a declaration file for this untyped library.
- 翻译：对于这个没有类型定义的库，我们可能需要写一个声明文件。
- 讲解：接第三方库时常见。

## 12. ambient declaration
- 发音：短语表达
- 含义：环境声明
- 例句：Ambient declarations are useful for global variables.
- 翻译：环境声明对全局变量很有用。
- 讲解：TS 配置与类型声明常见。

## 13. strict mode
- 发音：短语表达
- 含义：严格模式
- 例句：Strict mode catches many unsafe assumptions at compile time.
- 翻译：严格模式可以在编译期捕获很多不安全假设。
- 讲解：工程规范高频词。

## 14. non-null assertion
- 发音：短语表达
- 含义：非空断言
- 例句：Non-null assertions should be avoided unless you are completely sure.
- 翻译：除非你完全确定，否则应尽量避免非空断言。
- 讲解：很适合代码评审。

## 15. generic constraint
- 发音：短语表达
- 含义：泛型约束
- 例句：A generic constraint prevents invalid type arguments.
- 翻译：泛型约束可以防止无效的类型参数。
- 讲解：TS 泛型进阶表达。
