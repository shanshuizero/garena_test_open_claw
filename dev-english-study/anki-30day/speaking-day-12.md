# Speaking Day 12：Vue 核心

## 晨读短句
1. The reactive state is shared across components.
   - 中文：响应式的
   - 提示：Vue 核心思想

2. A computed property should be free of side effects.
   - 中文：计算属性
   - 提示：Vue 高频

3. Use a watcher only when a computed property is not enough.
   - 中文：侦听器
   - 提示：谨慎使用

4. Slots make the component API more flexible.
   - 中文：插槽
   - 提示：组件封装

5. A route guard can block unauthorized access.
   - 中文：路由守卫
   - 提示：权限控制
