# Anki Day 09：测试 / 质量 / 安全

## 卡片格式
Front｜Back

1. **Front**: unit test
   **Back**: 单元测试｜This function still needs a unit test.｜提示：测试基础

2. **Front**: integration test
   **Back**: 集成测试｜The integration test covers the full login flow.｜提示：联动测试

3. **Front**: regression
   **Back**: 回归问题｜We need a regression test for this bug.｜提示：防止旧问题回归

4. **Front**: vulnerability
   **Back**: 漏洞｜The scanner found a critical vulnerability.｜提示：安全高频

5. **Front**: secret
   **Back**: 敏感凭据｜Never hardcode secrets in the repository.｜提示：安全基本规则
