# Speaking Day 21：系统设计面试表达

## 晨读短句
1. The write path should remain simple and reliable.
   - 中文：写路径
   - 提示：读写分离

2. The read path can be optimized separately.
   - 中文：读路径
   - 提示：系统拆分

3. Capacity planning depends on traffic patterns.
   - 中文：容量规划
   - 提示：面试高频

4. Replicas should be distributed across different fault domains.
   - 中文：故障域
   - 提示：高可用设计

5. Graceful degradation is better than complete failure.
   - 中文：优雅降级
   - 提示：系统设计思维
