# Anki Day 25：书面表达：日报 / 周报

## 卡片格式
Front｜Back

1. **Front**: Today I worked on...
   **Back**: 今天我处理了……｜Today I worked on the user authentication flow.｜提示：日报模板

2. **Front**: Tomorrow I plan to...
   **Back**: 明天我计划……｜Tomorrow I plan to finish the remaining test cases.｜提示：日报模板

3. **Front**: This week I focused on...
   **Back**: 这周我主要在做……｜This week I focused on improving API stability.｜提示：周报模板

4. **Front**: The main achievements were...
   **Back**: 主要成果包括……｜The main achievements were reducing latency and adding caching.｜提示：周报模板

5. **Front**: The main challenge was...
   **Back**: 最大的挑战是……｜The main challenge was coordinating changes across teams.｜提示：周报模板
