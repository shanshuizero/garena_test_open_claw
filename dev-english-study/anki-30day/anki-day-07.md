# Anki Day 07：API / 数据库 / DevOps

## 卡片格式
Front｜Back

1. **Front**: request
   **Back**: 请求｜The request payload is missing a required field.｜提示：接口基础词

2. **Front**: response
   **Back**: 响应｜The response format has changed.｜提示：接口返回

3. **Front**: schema
   **Back**: 结构定义｜We need to update the schema.｜提示：数据库/API 结构

4. **Front**: migration
   **Back**: 迁移｜Run the migration before starting the application.｜提示：发布前常见动作

5. **Front**: pipeline
   **Back**: 流水线｜The pipeline failed at the integration test stage.｜提示：CI/CD 高频
