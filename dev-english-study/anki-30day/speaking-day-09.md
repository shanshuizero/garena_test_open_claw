# Speaking Day 09：测试 / 质量 / 安全

## 晨读短句
1. This function still needs a unit test.
   - 中文：单元测试
   - 提示：测试基础

2. The integration test covers the full login flow.
   - 中文：集成测试
   - 提示：联动测试

3. We need a regression test for this bug.
   - 中文：回归问题
   - 提示：防止旧问题回归

4. The scanner found a critical vulnerability.
   - 中文：漏洞
   - 提示：安全高频

5. Never hardcode secrets in the repository.
   - 中文：敏感凭据
   - 提示：安全基本规则
