# Speaking Day 18：Java 后端

## 晨读短句
1. Dependency injection improves testability.
   - 中文：依赖注入
   - 提示：Spring 核心

2. Transaction propagation affects nested service calls.
   - 中文：事务传播
   - 提示：事务高频

3. The thread pool is too small for the workload.
   - 中文：线程池
   - 提示：并发调优

4. Frequent garbage collection pauses can affect latency.
   - 中文：垃圾回收
   - 提示：JVM 调优

5. The interceptor checks authentication before the controller.
   - 中文：拦截器
   - 提示：请求链处理
