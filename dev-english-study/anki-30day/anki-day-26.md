# Anki Day 26：书面表达：PR / Issue / 会议纪要

## 卡片格式
Front｜Back

1. **Front**: This PR refactors...
   **Back**: 这个 PR 重构了……｜This PR refactors the notification module.｜提示：PR 描述

2. **Front**: The overall direction looks good to me.
   **Back**: 整体方向我觉得不错｜The overall direction looks good to me.｜提示：PR 评论

3. **Front**: Thanks for reporting this issue.
   **Back**: 感谢反馈这个问题｜Thanks for reporting this issue.｜提示：Issue 回复

4. **Front**: The root cause is currently under investigation.
   **Back**: 根因仍在排查中｜The root cause is currently under investigation.｜提示：Issue 回复

5. **Front**: Key decisions
   **Back**: 关键决定｜Key decisions: separate the write path from the delivery path.｜提示：会议纪要
