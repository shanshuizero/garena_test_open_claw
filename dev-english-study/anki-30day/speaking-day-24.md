# Speaking Day 24：需求评审 / PR / 故障沟通

## 晨读短句
1. From the frontend perspective, we need a stable response structure.
   - 中文：从前端角度来看……
   - 提示：需求评审

2. From the backend side, we need to confirm partial updates.
   - 中文：从后端角度来说……
   - 提示：需求评审

3. One concern is that this change may break older clients.
   - 中文：一个顾虑是……
   - 提示：PR 评论

4. We are seeing a sharp increase in error rate.
   - 中文：我们看到错误率快速上升
   - 提示：故障同步

5. Let us roll back first and continue investigating.
   - 中文：我们先回滚
   - 提示：事故处理

## 跟读提示
- 先慢读 2 遍
- 再正常语速读 3 遍
- 最后脱稿复述其中 2 句
