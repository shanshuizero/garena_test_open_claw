# Speaking Day 20：Node.js 后端

## 晨读短句
1. Node.js is a good fit for event-driven architecture.
   - 中文：事件驱动架构
   - 提示：Node 典型定位

2. Non-blocking I/O improves throughput.
   - 中文：非阻塞 I/O
   - 提示：Node 优势

3. Stream backpressure prevents memory usage from growing too fast.
   - 中文：流式背压
   - 提示：Stream 场景

4. The process crashes because of a memory leak.
   - 中文：内存泄漏
   - 提示：线上高频问题

5. CPU-intensive tasks should be moved to worker threads.
   - 中文：工作线程
   - 提示：CPU 密集任务
