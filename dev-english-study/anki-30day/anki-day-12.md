# Anki Day 12：Vue 核心

## 卡片格式
Front｜Back

1. **Front**: reactive
   **Back**: 响应式的｜The reactive state is shared across components.｜提示：Vue 核心思想

2. **Front**: computed property
   **Back**: 计算属性｜A computed property should be free of side effects.｜提示：Vue 高频

3. **Front**: watcher
   **Back**: 侦听器｜Use a watcher only when a computed property is not enough.｜提示：谨慎使用

4. **Front**: slot
   **Back**: 插槽｜Slots make the component API more flexible.｜提示：组件封装

5. **Front**: route guard
   **Back**: 路由守卫｜A route guard can block unauthorized access.｜提示：权限控制
