# Speaking Day 01：通用开发核心 1

## 晨读短句
1. Our architecture needs to support high concurrency.
   - 中文：架构
   - 提示：系统整体设计

2. We should split this feature into smaller modules.
   - 中文：模块
   - 提示：功能拆分单元

3. This component is reused across multiple pages.
   - 中文：组件
   - 提示：前端和系统都常用

4. Please update the dependency to the latest stable version.
   - 中文：依赖
   - 提示：依赖包/依赖关系

5. Check whether the database configuration is correct.
   - 中文：配置
   - 提示：配置项高频词
