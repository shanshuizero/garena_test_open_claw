# Speaking Day 11：React 核心

## 晨读短句
1. The virtual DOM helps reduce direct DOM manipulation.
   - 中文：虚拟 DOM
   - 提示：React 原理

2. Poor key usage can affect the reconciliation process.
   - 中文：协调更新
   - 提示：Diff 过程

3. A controlled component makes form validation easier.
   - 中文：受控组件
   - 提示：表单场景

4. Memoization can improve performance.
   - 中文：记忆化
   - 提示：性能优化

5. Fetching data inside render will cause side effects.
   - 中文：副作用
   - 提示：Hooks 核心概念
