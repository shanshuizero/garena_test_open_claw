# Speaking Day 13：TypeScript 核心

## 晨读短句
1. Type inference works well here.
   - 中文：类型推断
   - 提示：减少显式标注

2. This helper can be rewritten as a generic function.
   - 中文：泛型
   - 提示：复用类型逻辑

3. Type narrowing is required here.
   - 中文：类型收窄
   - 提示：缩小类型范围

4. A type guard makes the code safer.
   - 中文：类型守卫
   - 提示：运行时判断

5. Utility types can simplify model definitions.
   - 中文：工具类型
   - 提示：TS 高频
