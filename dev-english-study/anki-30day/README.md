# 30 天版：Anki 抽卡 + 晨读口语

这套资料把已有开发英语内容整理成：

- **Anki 抽卡版**：适合问答式记忆
- **晨读口语版**：适合每天 5～10 分钟朗读

## 目录结构

- `index.md`：30 天总目录
- `anki-day-01.md` ~ `anki-day-30.md`：每天一组抽卡
- `speaking-day-01.md` ~ `speaking-day-30.md`：每天一组晨读口语

## Anki 卡片格式

每张卡片统一采用：

- Front：英文 / 表达
- Back：中文 + 高频短句 + 简短提示

## 晨读口语格式

每份晨读材料包含：

- 今日主题
- 5～8 句高频朗读句
- 中文参考
- 跟读提示

## 建议节奏

- 12:30 收到当日内容
- 先看 Anki 抽卡 5 分钟
- 再朗读晨读口语 5 分钟
- 晚上可复习当天 1 次
