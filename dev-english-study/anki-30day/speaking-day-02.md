# Speaking Day 02：通用开发核心 2

## 晨读短句
1. I will implement the new permission logic today.
   - 中文：实现
   - 提示：实现功能/逻辑

2. We need to refactor this function.
   - 中文：重构
   - 提示：优化内部结构

3. We need a more scalable solution.
   - 中文：可扩展的
   - 提示：系统设计高频

4. The error handling is not robust enough.
   - 中文：健壮的
   - 提示：异常下仍稳定

5. We will deploy the patch tonight.
   - 中文：部署
   - 提示：发布部署高频

## 跟读提示
- 先慢读 2 遍
- 再正常语速读 3 遍
