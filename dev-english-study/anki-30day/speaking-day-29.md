# Speaking Day 29：综合复习 1

## 晨读短句
1. This change is backward compatible.
   - 中文：向后兼容
   - 提示：发布高频表达

2. The trade-off here is between simplicity and flexibility.
   - 中文：权衡
   - 提示：架构高频

3. This logic fails on several edge cases.
   - 中文：边界情况
   - 提示：测试/评审高频

4. We need a fallback when the API is unavailable.
   - 中文：兜底方案
   - 提示：稳定性高频

5. We need clear ownership for this service.
   - 中文：责任归属
   - 提示：协作高频
