# Speaking Day 17：数据库设计 / 索引 / 查询

## 晨读短句
1. Read replicas reduce the load on the primary database.
   - 中文：只读副本
   - 提示：读写分离

2. Denormalization can improve read performance.
   - 中文：反规范化
   - 提示：换性能的 trade-off

3. The query plan shows that the index is not being used.
   - 中文：查询计划
   - 提示：SQL 优化

4. A composite index may improve this query.
   - 中文：联合索引
   - 提示：多列查询

5. A covering index avoids extra table lookups.
   - 中文：覆盖索引
   - 提示：避免回表
