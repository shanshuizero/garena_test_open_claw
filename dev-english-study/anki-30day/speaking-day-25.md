# Speaking Day 25：书面表达：日报 / 周报

## 晨读短句
1. Today I worked on the user authentication flow.
   - 中文：今天我处理了……
   - 提示：日报模板

2. Tomorrow I plan to finish the remaining test cases.
   - 中文：明天我计划……
   - 提示：日报模板

3. This week I focused on improving API stability.
   - 中文：这周我主要在做……
   - 提示：周报模板

4. The main achievements were reducing latency and adding caching.
   - 中文：主要成果包括……
   - 提示：周报模板

5. The main challenge was coordinating changes across teams.
   - 中文：最大的挑战是……
   - 提示：周报模板

## 跟读提示
- 先慢读 2 遍
- 再正常语速读 3 遍
- 最后脱稿复述其中 2 句
