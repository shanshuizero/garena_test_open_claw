# Speaking Day 07：API / 数据库 / DevOps

## 晨读短句
1. The request payload is missing a required field.
   - 中文：请求
   - 提示：接口基础词

2. The response format has changed.
   - 中文：响应
   - 提示：接口返回

3. We need to update the schema.
   - 中文：结构定义
   - 提示：数据库/API 结构

4. Run the migration before starting the application.
   - 中文：迁移
   - 提示：发布前常见动作

5. The pipeline failed at the integration test stage.
   - 中文：流水线
   - 提示：CI/CD 高频

## 跟读提示
- 先慢读 2 遍
- 再正常语速读 3 遍
- 最后脱稿复述其中 2 句
