# Anki Day 02：通用开发核心 2

## 卡片格式
Front｜Back

1. **Front**: implement
   **Back**: 实现｜I will implement the new permission logic today.｜提示：实现功能/逻辑

2. **Front**: refactor
   **Back**: 重构｜We need to refactor this function.｜提示：优化内部结构

3. **Front**: scalable
   **Back**: 可扩展的｜We need a more scalable solution.｜提示：系统设计高频

4. **Front**: robust
   **Back**: 健壮的｜The error handling is not robust enough.｜提示：异常下仍稳定

5. **Front**: deploy
   **Back**: 部署｜We will deploy the patch tonight.｜提示：发布部署高频
