# Anki Day 05：后端基础 1

## 卡片格式
Front｜Back

1. **Front**: endpoint
   **Back**: 接口端点｜This endpoint returns the user profile data.｜提示：API 地址

2. **Front**: service
   **Back**: 服务｜The notification service is temporarily unavailable.｜提示：业务服务

3. **Front**: authentication
   **Back**: 认证｜Authentication fails when the token expires.｜提示：确认身份

4. **Front**: authorization
   **Back**: 授权｜Authorization should be enforced on the server side.｜提示：确认权限

5. **Front**: token
   **Back**: 令牌｜The access token is stored in memory.｜提示：JWT 场景高频
