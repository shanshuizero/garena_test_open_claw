# Anki Day 24：需求评审 / PR / 故障沟通

## 卡片格式
Front｜Back

1. **Front**: From the frontend perspective...
   **Back**: 从前端角度来看……｜From the frontend perspective, we need a stable response structure.｜提示：需求评审

2. **Front**: From the backend side...
   **Back**: 从后端角度来说……｜From the backend side, we need to confirm partial updates.｜提示：需求评审

3. **Front**: One concern is that...
   **Back**: 一个顾虑是……｜One concern is that this change may break older clients.｜提示：PR 评论

4. **Front**: We are seeing a sharp increase in error rate.
   **Back**: 我们看到错误率快速上升｜We are seeing a sharp increase in error rate.｜提示：故障同步

5. **Front**: Let us roll back first.
   **Back**: 我们先回滚｜Let us roll back first and continue investigating.｜提示：事故处理
