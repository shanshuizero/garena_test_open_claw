# Speaking Day 03：前端基础 1

## 晨读短句
1. The mobile layout breaks on small screens.
   - 中文：布局
   - 提示：页面排布

2. We need a responsive design for tablets.
   - 中文：响应式的
   - 提示：适配不同尺寸

3. The modal should stay centered in the viewport.
   - 中文：视口
   - 提示：浏览器可视区域

4. The page takes too long to render.
   - 中文：渲染
   - 提示：页面显示过程

5. The main bundle is too large.
   - 中文：打包产物
   - 提示：前端工程化高频

## 跟读提示
- 先慢读 2 遍
- 再正常语速读 3 遍
- 最后脱稿复述其中 2 句
