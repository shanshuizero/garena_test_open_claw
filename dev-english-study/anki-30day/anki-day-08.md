# Anki Day 08：调试 / 协作 / 评审

## 卡片格式
Front｜Back

1. **Front**: reproduce
   **Back**: 复现｜I can reproduce the bug on staging.｜提示：排障第一步

2. **Front**: diagnose
   **Back**: 诊断/排查｜It took us hours to diagnose the root cause.｜提示：定位问题

3. **Front**: root cause
   **Back**: 根因｜The root cause was a race condition.｜提示：问题本质原因

4. **Front**: clarify
   **Back**: 澄清｜Can you clarify the expected behavior?｜提示：需求讨论高频

5. **Front**: align
   **Back**: 对齐｜Let us align on the API contract first.｜提示：团队沟通高频
