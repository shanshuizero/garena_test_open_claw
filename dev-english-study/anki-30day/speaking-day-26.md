# Speaking Day 26：书面表达：PR / Issue / 会议纪要

## 晨读短句
1. This PR refactors the notification module.
   - 中文：这个 PR 重构了……
   - 提示：PR 描述

2. The overall direction looks good to me.
   - 中文：整体方向我觉得不错
   - 提示：PR 评论

3. Thanks for reporting this issue.
   - 中文：感谢反馈这个问题
   - 提示：Issue 回复

4. The root cause is currently under investigation.
   - 中文：根因仍在排查中
   - 提示：Issue 回复

5. Key decisions: separate the write path from the delivery path.
   - 中文：关键决定
   - 提示：会议纪要

## 跟读提示
- 先慢读 2 遍
- 再正常语速读 3 遍
- 最后脱稿复述其中 2 句
