# Anki Day 29：综合复习 1

## 卡片格式
Front｜Back

1. **Front**: backward compatible
   **Back**: 向后兼容｜This change is backward compatible.｜提示：发布高频表达

2. **Front**: trade-off
   **Back**: 权衡｜The trade-off here is between simplicity and flexibility.｜提示：架构高频

3. **Front**: edge case
   **Back**: 边界情况｜This logic fails on several edge cases.｜提示：测试/评审高频

4. **Front**: fallback
   **Back**: 兜底方案｜We need a fallback when the API is unavailable.｜提示：稳定性高频

5. **Front**: ownership
   **Back**: 责任归属｜We need clear ownership for this service.｜提示：协作高频
