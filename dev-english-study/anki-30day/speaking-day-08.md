# Speaking Day 08：调试 / 协作 / 评审

## 晨读短句
1. I can reproduce the bug on staging.
   - 中文：复现
   - 提示：排障第一步

2. It took us hours to diagnose the root cause.
   - 中文：诊断/排查
   - 提示：定位问题

3. The root cause was a race condition.
   - 中文：根因
   - 提示：问题本质原因

4. Can you clarify the expected behavior?
   - 中文：澄清
   - 提示：需求讨论高频

5. Let us align on the API contract first.
   - 中文：对齐
   - 提示：团队沟通高频

## 跟读提示
- 先慢读 2 遍
- 再正常语速读 3 遍
- 最后脱稿复述其中 2 句
