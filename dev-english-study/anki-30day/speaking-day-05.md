# Speaking Day 05：后端基础 1

## 晨读短句
1. This endpoint returns the user profile data.
   - 中文：接口端点
   - 提示：API 地址

2. The notification service is temporarily unavailable.
   - 中文：服务
   - 提示：业务服务

3. Authentication fails when the token expires.
   - 中文：认证
   - 提示：确认身份

4. Authorization should be enforced on the server side.
   - 中文：授权
   - 提示：确认权限

5. The access token is stored in memory.
   - 中文：令牌
   - 提示：JWT 场景高频

- 先慢读 2 遍
- 再正常语速读 3 遍
