# Speaking Day 06：后端基础 2

## 晨读短句
1. We should cache this result to reduce database pressure.
   - 中文：缓存
   - 提示：性能优化

2. The jobs are processed through a message queue.
   - 中文：队列
   - 提示：异步系统

3. The request fails because of a database timeout.
   - 中文：超时
   - 提示：线上排障

4. This design improves concurrency.
   - 中文：并发
   - 提示：系统设计核心

5. This callback must be idempotent.
   - 中文：幂等的
   - 提示：支付/消息高频
