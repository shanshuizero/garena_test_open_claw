# Anki Day 23：进阶工作对话

## 卡片格式
Front｜Back

1. **Front**: The coupling is too tight.
   **Back**: 耦合太紧了｜The coupling makes independent deployments difficult.｜提示：架构评审

2. **Front**: That would reduce the blast radius.
   **Back**: 那样会降低影响范围｜That would reduce the blast radius of future changes.｜提示：稳定性表达

3. **Front**: I am not convinced.
   **Back**: 我还不太认可｜I am not convinced it is the lowest-risk option.｜提示：分歧表达

4. **Front**: A safer path would be...
   **Back**: 更稳妥的路径是……｜A safer path would be an incremental refactor.｜提示：表达建议

5. **Front**: I would like to add one point.
   **Back**: 我想补充一点｜I would like to add one point from the backend perspective.｜提示：国际会议发言
