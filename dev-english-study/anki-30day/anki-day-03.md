# Anki Day 03：前端基础 1

## 卡片格式
Front｜Back

1. **Front**: layout
   **Back**: 布局｜The mobile layout breaks on small screens.｜提示：页面排布

2. **Front**: responsive
   **Back**: 响应式的｜We need a responsive design for tablets.｜提示：适配不同尺寸

3. **Front**: viewport
   **Back**: 视口｜The modal should stay centered in the viewport.｜提示：浏览器可视区域

4. **Front**: render
   **Back**: 渲染｜The page takes too long to render.｜提示：页面显示过程

5. **Front**: bundle
   **Back**: 打包产物｜The main bundle is too large.｜提示：前端工程化高频
