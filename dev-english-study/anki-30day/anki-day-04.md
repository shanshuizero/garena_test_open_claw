# Anki Day 04：前端基础 2

## 卡片格式
Front｜Back

1. **Front**: state
   **Back**: 状态｜The loading state is not handled properly.｜提示：组件/页面状态

2. **Front**: props
   **Back**: 属性｜Do not mutate props inside the child component.｜提示：React 高频

3. **Front**: hook
   **Back**: 钩子｜This custom hook simplifies the fetch logic.｜提示：React 高频

4. **Front**: debounce
   **Back**: 防抖｜We should debounce the search input.｜提示：输入优化

5. **Front**: lazy loading
   **Back**: 懒加载｜Lazy loading reduces the initial load time.｜提示：按需加载
