# 开发英语学习资料总目录

这是一套面向前端、后端与通用软件工程场景的专业英语学习资料。

## 文件结构

- `00-word-index.md`：总词汇目录，按主题归类，便于检索与背诵
- `01-core-development-english.md`：通用开发英语核心词汇
- `02-frontend-english.md`：前端开发英语词汇
- `03-backend-english.md`：后端开发英语词汇
- `04-api-database-devops-english.md`：API、数据库、DevOps 与部署英语词汇
- `05-meeting-pr-review-debug-english.md`：代码评审、调试、协作、会议表达

## 每个词条包含内容

- 英文单词 / 短语
- 音标或近似发音提示
- 词义讲解
- 开发语境说明
- 例句
- 例句翻译
- 句子讲解

## 学习建议

1. 先读 `00-word-index.md`，建立整体地图。
2. 再按岗位重点学习：
   - 偏前端：优先看 `02-frontend-english.md`
   - 偏后端：优先看 `03-backend-english.md`
   - 偏全栈：全部都看，重点补 `04-api-database-devops-english.md`
3. 把例句改写成你项目里的真实句子，效果最好。
4. 面试或工作沟通前，重点复习 `05-meeting-pr-review-debug-english.md`。
