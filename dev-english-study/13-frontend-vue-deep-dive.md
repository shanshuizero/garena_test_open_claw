# 前端深入：Vue 专项英语

## 1. Composition API
- 发音：短语表达
- 含义：组合式 API
- 例句：The Composition API improves logic reuse in large components.
- 翻译：组合式 API 提升了大型组件中的逻辑复用能力。
- 讲解：Vue 3 核心概念。

## 2. Options API
- 发音：短语表达
- 含义：选项式 API
- 例句：The Options API is easier for beginners to understand.
- 翻译：选项式 API 对初学者更容易理解。
- 讲解：常用于和 Composition API 对比。

## 3. ref
- 发音：/ref/
- 含义：响应式引用
- 例句：Use a ref when you need a reactive primitive value.
- 翻译：当你需要一个响应式的基础类型值时，使用 ref。
- 讲解：Vue 3 高频概念。

## 4. reactive object
- 发音：短语表达
- 含义：响应式对象
- 例句：A reactive object is useful for grouped state.
- 翻译：响应式对象适合管理成组状态。
- 讲解：与 ref 配合记忆更好。

## 5. computed caching
- 发音：短语表达
- 含义：计算属性缓存
- 例句：Computed caching avoids unnecessary recomputation.
- 翻译：计算属性缓存可以避免不必要的重复计算。
- 讲解：Vue 性能优化基础点。

## 6. watchEffect
- 发音：短语表达
- 含义：自动追踪依赖的侦听
- 例句：watchEffect is convenient, but explicit watchers are sometimes clearer.
- 翻译：watchEffect 很方便，但有时显式 watcher 更清晰。
- 讲解：很符合真实代码评审场景。

## 7. slot
- 发音：/slɑːt/
- 含义：插槽
- 例句：Slots make the component API more flexible.
- 翻译：插槽让组件 API 更灵活。
- 讲解：组件封装高频术语。

## 8. scoped slot
- 发音：短语表达
- 含义：作用域插槽
- 例句：A scoped slot allows the parent to access child data.
- 翻译：作用域插槽允许父组件访问子组件数据。
- 讲解：Vue 组件通信常见知识点。

## 9. directive
- 发音：/dəˈrektɪv/
- 含义：指令
- 例句：Custom directives should be used sparingly.
- 翻译：自定义指令应该谨慎使用。
- 讲解：很适合评审表达。

## 10. Pinia store
- 发音：短语表达
- 含义：Pinia 状态仓库
- 例句：We should move this shared logic into a Pinia store.
- 翻译：我们应该把这段共享逻辑迁移到 Pinia store 中。
- 讲解：Vue 状态管理高频表达。

## 11. route guard
- 发音：短语表达
- 含义：路由守卫
- 例句：A route guard can block unauthorized access.
- 翻译：路由守卫可以阻止未授权访问。
- 讲解：登录权限常见场景。

## 12. keep-alive
- 发音：短语表达
- 含义：组件缓存
- 例句：keep-alive helps preserve component state across route changes.
- 翻译：keep-alive 有助于在路由切换时保留组件状态。
- 讲解：实际项目非常常见。

## 13. transition group
- 发音：短语表达
- 含义：列表过渡组件
- 例句：A transition group can animate list insertions and removals.
- 翻译：transition group 可以为列表插入和删除添加动画。
- 讲解：UI 细节优化常见。

## 14. template ref
- 发音：短语表达
- 含义：模板引用
- 例句：Use a template ref only when direct DOM access is necessary.
- 翻译：只有在确实需要直接访问 DOM 时才使用模板引用。
- 讲解：是很好的最佳实践表达。

## 15. single-file component
- 发音：短语表达
- 含义：单文件组件
- 例句：A single-file component keeps template, script, and style together.
- 翻译：单文件组件把模板、脚本和样式放在一起。
- 讲解：Vue 基础术语之一。
