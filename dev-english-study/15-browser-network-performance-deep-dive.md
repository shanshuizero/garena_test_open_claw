# 前端深入：浏览器、网络与性能专项英语

## 1. DNS lookup
- 发音：短语表达
- 含义：DNS 查询
- 例句：The DNS lookup time is higher than expected.
- 翻译：DNS 查询时间比预期更高。
- 讲解：网络性能分析基础术语。

## 2. TCP handshake
- 发音：短语表达
- 含义：TCP 握手
- 例句：A slow TCP handshake can delay the first request.
- 翻译：TCP 握手过慢会延迟首个请求。
- 讲解：前端网络层分析常见。

## 3. TLS negotiation
- 发音：短语表达
- 含义：TLS 协商
- 例句：TLS negotiation adds extra latency to secure connections.
- 翻译：TLS 协商会给安全连接增加额外延迟。
- 讲解：HTTPS 性能讨论常见。

## 4. cache-control
- 发音：短语表达
- 含义：缓存控制头
- 例句：The cache-control header is too conservative.
- 翻译：这个缓存控制头设置得过于保守。
- 讲解：Web 缓存优化很常见。

## 5. cache hit ratio
- 发音：短语表达
- 含义：缓存命中率
- 例句：A higher cache hit ratio reduces origin traffic.
- 翻译：更高的缓存命中率可以减少源站流量。
- 讲解：CDN 和静态资源优化高频表达。

## 6. content delivery network
- 发音：短语表达
- 含义：内容分发网络
- 例句：A content delivery network improves latency for global users.
- 翻译：内容分发网络可以改善全球用户的访问延迟。
- 讲解：CDN 全称。

## 7. resource hint
- 发音：短语表达
- 含义：资源提示
- 例句：Resource hints can improve the loading priority of critical assets.
- 翻译：资源提示可以改善关键资源的加载优先级。
- 讲解：prefetch/preload/preconnect 都属于这类概念。

## 8. preconnect
- 发音：/ˌpriːkəˈnekt/
- 含义：预连接
- 例句：We should preconnect to the font host.
- 翻译：我们应该预连接到字体资源主机。
- 讲解：首屏优化常见。

## 9. time to first byte
- 发音：短语表达
- 含义：首字节时间
- 例句：The time to first byte is affected by server latency.
- 翻译：首字节时间会受到服务端延迟影响。
- 讲解：TTFB 是性能术语。

## 10. first contentful paint
- 发音：短语表达
- 含义：首次内容绘制
- 例句：We improved the first contentful paint by reducing render-blocking resources.
- 翻译：我们通过减少阻塞渲染的资源改善了首次内容绘制。
- 讲解：Web Vitals 相关。

## 11. largest contentful paint
- 发音：短语表达
- 含义：最大内容绘制
- 例句：The hero image is hurting the largest contentful paint.
- 翻译：首屏主图拖慢了最大内容绘制指标。
- 讲解：Core Web Vitals 核心指标。

## 12. cumulative layout shift
- 发音：短语表达
- 含义：累积布局偏移
- 例句：Missing image dimensions can increase cumulative layout shift.
- 翻译：缺少图片尺寸会增加累积布局偏移。
- 讲解：CLS 高频原因。

## 13. interaction to next paint
- 发音：短语表达
- 含义：下一次绘制交互延迟
- 例句：Long main-thread tasks hurt interaction to next paint.
- 翻译：主线程长任务会损害交互到下一次绘制的指标。
- 讲解：INP 是新的重要交互指标。

## 14. render-blocking resource
- 发音：短语表达
- 含义：阻塞渲染资源
- 例句：Too many render-blocking resources slow down the initial render.
- 翻译：过多的阻塞渲染资源会拖慢初始渲染。
- 讲解：前端优化高频表达。

## 15. main thread bottleneck
- 发音：短语表达
- 含义：主线程瓶颈
- 例句：Heavy JavaScript execution creates a main thread bottleneck.
- 翻译：大量 JavaScript 执行会造成主线程瓶颈。
- 讲解：性能分析中的典型说法。
