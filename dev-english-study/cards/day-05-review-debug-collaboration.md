# Day 05：评审 / 调试 / 协作表达

## 1. reproduce
- 发音：/ˌriːprəˈduːs/
- 中文：复现
- 高频短句：I can reproduce the bug on staging.

## 2. diagnose
- 发音：/ˈdaɪəɡnoʊs/
- 中文：诊断 / 排查
- 高频短句：It took us hours to diagnose the root cause.

## 3. root cause
- 发音：/ruːt kɔːz/
- 中文：根因
- 高频短句：The root cause was a race condition.

## 4. workaround
- 发音：/ˈwɜːrkaˌraʊnd/
- 中文：临时方案
- 高频短句：We have a workaround for now.

## 5. review
- 发音：/rɪˈvjuː/
- 中文：评审
- 高频短句：Could you review this pull request?

## 6. suggestion
- 发音：/səˈdʒestʃən/
- 中文：建议
- 高频短句：I have a small suggestion regarding the naming.

## 7. concern
- 发音：/kənˈsɜːrn/
- 中文：顾虑
- 高频短句：My main concern is the impact on existing users.

## 8. blocking
- 发音：/ˈblɑːkɪŋ/
- 中文：阻塞性的
- 高频短句：This is a blocking issue for the release.

## 9. clarify
- 发音：/ˈklærəfaɪ/
- 中文：澄清
- 高频短句：Can you clarify the expected behavior?

## 10. align
- 发音：/əˈlaɪn/
- 中文：对齐
- 高频短句：Let us align on the API contract first.

## 11. estimate
- 发音：/ˈestɪmeɪt/
- 中文：评估
- 高频短句：It is hard to estimate the effort.

## 12. ownership
- 发音：/ˈoʊnərʃɪp/
- 中文：责任归属
- 高频短句：We need clear ownership for this service.
