# Day 06：测试 / 质量 / 安全

## 1. test case
- 发音：/test keɪs/
- 中文：测试用例
- 高频短句：Please add more test cases for invalid input.

## 2. unit test
- 发音：/ˈjuːnɪt test/
- 中文：单元测试
- 高频短句：This function still needs a unit test.

## 3. integration test
- 发音：/ˌɪntɪˈɡreɪʃən test/
- 中文：集成测试
- 高频短句：The integration test covers the full login flow.

## 4. regression
- 发音：/rɪˈɡreʃən/
- 中文：回归问题
- 高频短句：We need a regression test for this bug.

## 5. flaky
- 发音：/ˈfleɪki/
- 中文：不稳定的
- 高频短句：This test is flaky.

## 6. vulnerability
- 发音：/ˌvʌlnərəˈbɪləti/
- 中文：漏洞
- 高频短句：The scanner found a critical vulnerability.

## 7. encryption
- 发音：/ɪnˈkrɪpʃən/
- 中文：加密
- 高频短句：Sensitive data must be protected with encryption.

## 8. secret
- 发音：/ˈsiːkrət/
- 中文：敏感凭据
- 高频短句：Never hardcode secrets in the repository.

## 9. sanitize
- 发音：/ˈsænɪtaɪz/
- 中文：净化 / 清洗
- 高频短句：User-generated content must be sanitized before rendering.

## 10. audit log
- 发音：/ˈɔːdɪt lɔːɡ/
- 中文：审计日志
- 高频短句：Every admin action should be recorded in the audit log.
