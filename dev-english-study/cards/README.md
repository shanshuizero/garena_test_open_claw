# 开发英语背诵卡片

这是一套从现有资料整理出来的“每日背诵版”卡片。

## 使用方式

- 每天学习 1 份卡片
- 每份建议 10～15 个词 / 表达
- 学习顺序：
  1. 先看英文和发音
  2. 再看中文意思
  3. 再读例句
  4. 最后遮住中文尝试复述

## 卡片格式

每张卡片统一包含：
- 英文
- 发音
- 中文
- 高频短句
- 记忆提示

## 建议节奏

- 第 1 遍：理解
- 第 2 遍：朗读
- 第 3 遍：遮挡复述
- 第 4 遍：自己造句
