# Day 10：分布式 / 数据库 / 消息系统

## 1. microservice
- 发音：/ˈmaɪkroʊˌsɜːrvɪs/
- 中文：微服务
- 高频短句：Each microservice should have a clear business boundary.

## 2. service contract
- 发音：短语表达
- 中文：服务契约
- 高频短句：Breaking the service contract will affect downstream teams.

## 3. backpressure
- 发音：/ˈbækˌpreʃər/
- 中文：背压
- 高频短句：We need backpressure to protect consumers.

## 4. dead letter queue
- 发音：短语表达
- 中文：死信队列
- 高频短句：Failed messages are routed to the dead letter queue.

## 5. consumer group
- 发音：短语表达
- 中文：消费者组
- 高频短句：Each partition is assigned to one consumer group member.

## 6. partition key
- 发音：短语表达
- 中文：分区键
- 高频短句：Choose the partition key carefully.

## 7. read replica
- 发音：短语表达
- 中文：只读副本
- 高频短句：Read replicas reduce the load on the primary database.

## 8. query plan
- 发音：短语表达
- 中文：查询计划
- 高频短句：The query plan shows that the index is not being used.

## 9. race condition
- 发音：短语表达
- 中文：竞态条件
- 高频短句：The issue is caused by a race condition.

## 10. quorum
- 发音：/ˈkwɔːrəm/
- 中文：多数仲裁
- 高频短句：The write succeeds only when a quorum acknowledges it.
