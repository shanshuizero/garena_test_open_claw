# Day 03：后端基础词汇

## 1. endpoint
- 发音：/ˈendpɔɪnt/
- 中文：接口端点
- 高频短句：This endpoint returns the user profile data.
- 记忆提示：API 地址。

## 2. service
- 发音：/ˈsɜːrvɪs/
- 中文：服务
- 高频短句：The notification service is temporarily unavailable.
- 记忆提示：业务服务、微服务都可用。

## 3. authentication
- 发音：/əˌθentɪˈkeɪʃən/
- 中文：认证
- 高频短句：Authentication fails when the token expires.
- 记忆提示：确认“你是谁”。

## 4. authorization
- 发音：/ˌɔːθərəˈzeɪʃən/
- 中文：授权
- 高频短句：Authorization should be enforced on the server side.
- 记忆提示：确认“你能做什么”。

## 5. token
- 发音：/ˈtoʊkən/
- 中文：令牌
- 高频短句：The access token is stored in memory.
- 记忆提示：JWT 场景高频。

## 6. cache
- 发音：/kæʃ/
- 中文：缓存
- 高频短句：We should cache this result to reduce database pressure.
- 记忆提示：性能优化核心词。

## 7. queue
- 发音：/kjuː/
- 中文：队列
- 高频短句：The jobs are processed through a message queue.
- 记忆提示：异步系统必备。

## 8. timeout
- 发音：/ˈtaɪmaʊt/
- 中文：超时
- 高频短句：The request fails because of a database timeout.
- 记忆提示：线上故障高频词。

## 9. concurrency
- 发音：/kənˈkɜːrənsi/
- 中文：并发
- 高频短句：This design improves concurrency.
- 记忆提示：系统设计核心概念。

## 10. transaction
- 发音：/trænˈzækʃən/
- 中文：事务
- 高频短句：The transaction should be rolled back if any step fails.
- 记忆提示：数据库一致性基础。

## 11. latency
- 发音：/ˈleɪtənsi/
- 中文：延迟
- 高频短句：The average latency is acceptable.
- 记忆提示：接口性能指标。

## 12. idempotent
- 发音：/ˌaɪdəmˈpoʊtənt/
- 中文：幂等的
- 高频短句：This callback must be idempotent.
- 记忆提示：支付、消息消费高频。
