# Day 02：前端基础词汇

## 1. layout
- 发音：/ˈleɪaʊt/
- 中文：布局
- 高频短句：The mobile layout breaks on small screens.
- 记忆提示：页面结构和排布。

## 2. responsive
- 发音：/rɪˈspɑːnsɪv/
- 中文：响应式的
- 高频短句：We need a responsive design for tablets.
- 记忆提示：适配不同屏幕尺寸。

## 3. viewport
- 发音：/ˈvjuːpɔːrt/
- 中文：视口
- 高频短句：The modal should stay centered in the viewport.
- 记忆提示：浏览器可见区域。

## 4. render
- 发音：/ˈrendər/
- 中文：渲染
- 高频短句：The page takes too long to render.
- 记忆提示：页面显示过程。

## 5. bundle
- 发音：/ˈbʌndl/
- 中文：打包产物
- 高频短句：The main bundle is too large.
- 记忆提示：Webpack / Vite 场景高频词。

## 6. asset
- 发音：/ˈæset/
- 中文：静态资源
- 高频短句：Static assets should be cached aggressively.
- 记忆提示：图片、字体、脚本都属于 asset。

## 7. selector
- 发音：/sɪˈlektər/
- 中文：选择器
- 高频短句：This selector is too specific.
- 记忆提示：CSS 高频词。

## 8. state
- 发音：/steɪt/
- 中文：状态
- 高频短句：The loading state is not handled properly.
- 记忆提示：组件状态、页面状态。

## 9. props
- 发音：/prɑːps/
- 中文：属性
- 高频短句：Do not mutate props inside the child component.
- 记忆提示：React 必备词。

## 10. hook
- 发音：/hʊk/
- 中文：钩子
- 高频短句：This custom hook simplifies the fetch logic.
- 记忆提示：React 项目高频。

## 11. debounce
- 发音：/diːˈbaʊns/
- 中文：防抖
- 高频短句：We should debounce the search input.
- 记忆提示：输入框高频优化。

## 12. lazy loading
- 发音：/ˈleɪzi ˈloʊdɪŋ/
- 中文：懒加载
- 高频短句：Lazy loading reduces the initial load time.
- 记忆提示：按需加载资源。
