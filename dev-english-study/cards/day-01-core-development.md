# Day 01：通用开发核心词汇

## 1. architecture
- 发音：/ˈɑːrkɪtektʃər/
- 中文：架构
- 高频短句：Our architecture needs to support high concurrency.
- 记忆提示：和系统整体设计有关。

## 2. module
- 发音：/ˈmɑːdʒuːl/
- 中文：模块
- 高频短句：We should split this feature into smaller modules.
- 记忆提示：表示功能拆分后的单元。

## 3. component
- 发音：/kəmˈpoʊnənt/
- 中文：组件
- 高频短句：This component is reused across multiple pages.
- 记忆提示：前端 UI 和系统组成部分都能用。

## 4. dependency
- 发音：/dɪˈpendənsi/
- 中文：依赖
- 高频短句：Please update the dependency to the latest stable version.
- 记忆提示：项目依赖包、模块依赖关系都可表示。

## 5. configuration
- 发音：/kənˌfɪɡjəˈreɪʃən/
- 中文：配置
- 高频短句：Check whether the database configuration is correct.
- 记忆提示：配置文件、配置项常见词。

## 6. implement
- 发音：/ˈɪmplɪment/
- 中文：实现
- 高频短句：I will implement the new permission logic today.
- 记忆提示：功能、接口、逻辑的实现。

## 7. refactor
- 发音：/riːˈfæktər/
- 中文：重构
- 高频短句：We need to refactor this function before adding new features.
- 记忆提示：不改外部行为，优化内部结构。

## 8. scalable
- 发音：/ˈskeɪləbl/
- 中文：可扩展的
- 高频短句：We need a more scalable solution.
- 记忆提示：常用于系统设计。

## 9. robust
- 发音：/roʊˈbʌst/
- 中文：健壮的
- 高频短句：The error handling is not robust enough.
- 记忆提示：异常情况下也稳定。

## 10. compatible
- 发音：/kəmˈpætəbl/
- 中文：兼容的
- 高频短句：The new API is backward compatible.
- 记忆提示：版本升级时的高频词。

## 11. deploy
- 发音：/dɪˈplɔɪ/
- 中文：部署
- 高频短句：We will deploy the patch tonight.
- 记忆提示：测试环境、生产环境都常用。

## 12. optimize
- 发音：/ˈɑːptɪmaɪz/
- 中文：优化
- 高频短句：We should optimize the loading strategy.
- 记忆提示：性能优化、结构优化都能用。
