# 后端深入：Java 后端专项英语

## 1. dependency injection
- 发音：短语表达
- 含义：依赖注入
- 例句：Dependency injection improves testability and modularity.
- 翻译：依赖注入提升了可测试性和模块化程度。
- 讲解：Spring 核心概念。

## 2. bean lifecycle
- 发音：短语表达
- 含义：Bean 生命周期
- 例句：Understanding the bean lifecycle helps avoid initialization issues.
- 翻译：理解 Bean 生命周期有助于避免初始化问题。
- 讲解：Spring 进阶高频点。

## 3. annotation-driven
- 发音：短语表达
- 含义：基于注解驱动的
- 例句：The framework is heavily annotation-driven.
- 翻译：这个框架非常依赖注解驱动。
- 讲解：Spring 生态常见表达。

## 4. transaction propagation
- 发音：短语表达
- 含义：事务传播
- 例句：Transaction propagation affects how nested service calls behave.
- 翻译：事务传播会影响嵌套服务调用的行为。
- 讲解：Spring 事务高频面试点。

## 5. isolation level
- 发音：短语表达
- 含义：隔离级别

- 翻译：选择合适的隔离级别是在一致性和性能之间做权衡。
- 讲解：数据库与事务核心概念。

## 6. lazy initialization
- 发音：短语表达
- 含义：延迟初始化
- 例句：Lazy initialization can reduce startup time.
- 翻译：延迟初始化可以减少启动时间。
- 讲解：框架启动优化常见。

## 7. eager loading
- 发音：短语表达
- 含义：急加载
- 例句：Eager loading may fetch unnecessary data.
- 翻译：急加载可能会拉取不必要的数据。

## 8. N+1 query problem
- 发音：短语表达
- 含义：N+1 查询问题
- 例句：The ORM layer suffers from an N+1 query problem.
- 翻译：ORM 层存在 N+1 查询问题。

## 9. thread pool

- 翻译：这个线程池对当前负载来说太小了。

- 翻译：频繁的垃圾回收停顿会影响延迟。
- 讲解：JVM 调优高频话题。

## 11. heap dump
- 发音：短语表达

## 12. classpath
- 发音：/ˈklæsˌpæθ/

- 讲解：Java 环境问题常见。

## 13. servlet filter

## 14. interceptor

- 翻译：拦截器会在请求到达控制器之前检查认证。
- 讲解：Spring MVC 高频术语。

## 15. autowiring
- 发音：/ˌɔːtoʊˈwaɪərɪŋ/
- 含义：自动注入
- 例句：Autowiring is convenient, but explicit configuration can be clearer in complex modules.
- 翻译：自动注入很方便，但在复杂模块中显式配置可能更清晰。
- 讲解：体现工程 trade-off。
